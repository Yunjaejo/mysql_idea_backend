### IDEA
사용자끼리 인테리어 소품을 소개해주는 공간입니다.<br>
팀원들의 아이디어, 그리고 IKEA의 서비스에서 영감을 얻었기에 <br>
프로젝트 명을 IDEA라고 정했습니다.

<hr>

### 소개
* 프론트엔드(React)와 백엔드(Node.js)의 첫 협업 프로젝트<br>
* CRUD를 이용한 회원가입, 게시글과 댓글, 위시리스트 구현
* ORM을 없이 데이터베이스 사용
